URL Shortener

  • Pass a URL as a parameter and receive a shortened URL in the response.
  • Visiting the shortened link redirects to the original link.
  • Takes URLs in the forms:
    • https://www.google.com
    • wwww.google.com
    • google.com
    • http://foo.com:8000
    • www.example.com/page1/page2

Example:

https://little-url.herokuapp.com/new/https://www.google.com

Returns

{ "original_url":"http://foo.com:80", "short_url":"https://little-url.herokuapp.com/8170" }

Usage:

https://little-url.herokuapp.com/2871 redirects to https://www.google.com
